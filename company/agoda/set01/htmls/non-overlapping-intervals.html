Non-overlapping Intervals](https://leetcode.com/problems/non-overlapping-intervals/) Medium Topics ![Image 6: icon](https://leetcode.com/_next/static/images/lock-a6627e2c7fa0ce8bc117c109fb4e567d.svg)Companies Given an array of intervals `intervals` where `intervals[i] = [starti, endi]`, return _the minimum number of intervals you need to remove to make the rest of the intervals non-overlapping_. **Note** that intervals which only touch at a point are **non-overlapping**. For example, `[1, 2]` and `[2, 3]` are non-overlapping. **Example 1:** **Input:** intervals = [[1,2],[2,3],[3,4],[1,3]] **Output:** 1 **Explanation:** [1,3] can be removed and the rest of the intervals are non-overlapping. **Example 2:** **Input:** intervals = [[1,2],[1,2],[1,2]] **Output:** 2 **Explanation:** You need to remove two [1,2] to make the rest of the intervals non-overlapping. **Example 3:** **Input:** intervals = [[1,2],[2,3]] **Output:** 0 **Explanation:** You don't need to remove any of the intervals since they're already non-overlapping. **Constraints:** * `1 <= intervals.length <= 105` * `intervals[i].length == 2` * `-5 * 104 <= starti < endi <= 5 * 104` Accepted 906,556/1.6M Acceptance Rate 56.3% * * * Topics [Array](https://leetcode.com/tag/array/)[Dynamic Programming](https://leetcode.com/tag/dynamic-programming/)[Greedy](https://leetcode.com/tag/greedy/)[Sorting](https://leetcode.com/tag/sorting/) * * * ![Image 7: icon](https://leetcode.com/_next/static/images/lock-a6627e2c7fa0ce8bc117c109fb4e567d.svg) Companies * * * Similar Questions [Minimum Number of Arrows to Burst Balloons](https://leetcode.com/problems/minimum-number-of-arrows-to-burst-balloons/) Medium [Determine if Two Events Have Conflict](https://leetcode.com/problems/determine-if-two-events-have-conflict/) Easy * * * Discussion (143) Choose a type Comment ðŸ’¡ Discussion Rules 1.
